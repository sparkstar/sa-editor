Editor Help

Workflow

  • Upload a translation JSON, select an active row, edit the EN or KO translation, then download the edited JSON.
  • Single line breaks in the editor are saved as [NL]. Blank lines are saved as [NEXT].
  • Rows marked as unused stay in the JSON. They are hidden from active filters, but can be restored with the Unused only filter.
  • Control-code warnings compare the JP original with each translation. 0x01-0x04, 0x0A, and 0x0D are ignored for this check.
  • Draft changes are autosaved to IndexedDB as a diff from the uploaded JSON.

Shortcuts

Focus EN translation.
Ctrl + Alt + K
Focus KO translation.
Ctrl + ]
Select the next visible row.
Ctrl + [
Select the previous visible row.
